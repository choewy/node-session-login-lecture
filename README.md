# node-session-login-lecture

## database

- MySQL(docker container)
- PORT : 33062

```
$ docker-compose up -d
```

## server

- Node.js(express)
- [setting](https://github.com/choewy/node-session-login-lecture/tree/master/server)
- run

```
$ cd server
$ npm install
$ npm run start
```

## client

- React.js
- run

```
$ cd client
$ npm install
$ npm run start
```
